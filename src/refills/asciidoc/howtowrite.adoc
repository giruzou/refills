== 定義リスト ― Definition List

[source,asciidoc]
.1行で記述
----
first term:: definition of first term
second term:: definition of second term
----

first term:: definition of first term
second term:: definition of second term


[source,asciidoc]
.複数行で記述
----
first term::
definition of first term
section term::
definition of second term
----

first term::
definition of first term
section term::
definition of second term
