== Array

=== 重複の除去 (unique / distinct)

- link:https://qiita.com/waka-ka/items/694866bd1a6a90289c33[[Javascript\] 配列の重複を取り出す方法 - Qiita]

```js
const animals = ["cat", "cat", "dog", "mouse", "dog"];
const distinctAnimals = [...new Set(animals)];
//=> ["cat", "dog", "mouse"]
```
